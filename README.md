## ✨ Features 
- 📝 post your talk
- 📚 read others' talk
- 💬 comment on others' talk
- ⏱️  real time interaction
- 📩 multiple file upload (image, video)
- ⚙️  custom your profile
- 📦 share your talk
- 📫 talk notification
- 📒 saving talk to your bookmark list
- 🔁 share your talk to your friends
- 🔥 trending 
- 🔎 exploring your thoughts
